title: "panguard whoami"
description: "Display the currently authenticated user and account details."
icon: "id-card"
---

The `whoami` command shows information about the currently logged-in user, including email, organization, and token expiry.

## Usage

```bash
panguard whoami [options]
```

## Options

<ParamField path="--json" type="flag">
  Output user information as structured JSON for scripting and automation.
</ParamField>

<ParamField path="--lang" type="string" default="en">
  Set the output language.
</ParamField>

## Examples

<CodeGroup>
```bash Show current user
panguard whoami
```

```bash JSON output for scripting
panguard whoami --json
```
</CodeGroup>
