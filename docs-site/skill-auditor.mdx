Panguard Skill Auditor scans third-party MCP skills for prompt injection, tool poisoning, hidden Unicode, encoded payloads, and other threats -- before they reach your agents.

## How It Works

```
panguard audit skill /path/to/skill-directory
```

The auditor analyzes the skill's `SKILL.md` and produces a risk score (0-100) with detailed findings.

## The Flywheel

Every skill audit contributes to the community:

1. **Audit** -- Scan a skill locally for threats
2. **Report** -- Anonymized findings are shared with Threat Cloud
3. **Review** -- Community voting + LLM analysis confirms threats
4. **Rules** -- Confirmed threats become ATR rules
5. **Protect** -- New rules strengthen the next audit
