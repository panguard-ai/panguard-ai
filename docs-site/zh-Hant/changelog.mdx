## v1.4.2

- 審計步驟現在使用 `--timeout` 和 `--no-cloud` 確保 CI 確定性

## v1.4.0

**發佈日期：2026-03-25** -- 16 個平台、OWASP 10/10、71 條 ATR 規則

### 16 個 AI 平台支援

- **平台覆蓋率翻倍**，從 8 個增至 16 個平台
- 新增平台：NemoClaw、ArkClaw、Cline、VS Code Copilot、Zed、Gemini CLI、Continue、Roo Code
- 119 個平台沙箱測試通過
- `panguard setup` 自動偵測並設定所有 16 個平台

### ATR v0.4.0

- **71 條 ATR 規則**（原 61 條），涵蓋 10 個新攻擊類別
- 新規則：eval 注入、shell 逃逸、動態 import、憑證竊取、OAuth token 濫用、環境變數收割、A2A 訊息驗證、Agent 身份偽造、審批疲勞、透過 Agent 的社交工程
- PINT 基準：62.7% 召回率、99.7% 精確率
- **OWASP Agentic Top 10：10/10 完整覆蓋**（77 個規則映射）

### Threat Cloud 即時指標

- `/api/metrics` 端點，提供所有來源（批量、CLI、Web）的即時掃描統計
- 網站數據現在從 Threat Cloud 即時拉取，靜態數據作為後備
- 標章 API：`GET /api/badge/:author/:skill` 用於「ATR Scanned」標章

## v1.3.3

**發佈日期：2026-03-25** -- 掃描準確度、`pga` 快捷鍵、新手引導

### 掃描準確度

- **大幅降低誤報。** 正常 Skills（weather、github、slack、notion）不再被標為 MEDIUM/HIGH。權限檢查現在會先移除程式碼區塊和否定段落再匹配。
- **新增隱匿攻擊偵測。** `<IMPORTANT>` 區塊攻擊和靜默資料外洩指令現在被標為 CRITICAL。
- **情境感知評分。** 已知 CLI 工具（curl、git、wget）和 API 整合獲得適當風險降低。憑證檢查區分竊取與正當使用。
- **13 個注入模式**（原 11 個），雙重匹配自動降級僅出現在程式碼範例中的 tool poisoning 模式。

### `pga` 快捷鍵

- **`pga` 指令** -- 三個字元從任何地方呼叫 Panguard。與 `panguard` 完全相同。
- **`pga up`** -- 一個指令啟動保護 + 儀表板。取代 `panguard guard start --dashboard`。
- **首次執行自動偵測** -- 第一次執行 `pga` 時自動啟動設定精靈。

### 新手引導

- **Guard 歡迎指南** -- Guard 首次啟動時顯示所有指令和 Guard 功能。
- **升級後顯示新功能** -- 更新後首次執行 CLI 顯示更新日誌重點。
- **乾淨的審計輸出** -- 非 verbose 模式不再輸出 JSON 日誌行。

## v0.8.0

**發佈日期：2026-03-13** -- 儀表板、AI 平台整合與技能審計器

### 內建儀表板

- 本地 Web 儀表板 `http://127.0.0.1:9100`，透過 `panguard guard start --dashboard` 啟動
- 7 個頁面：總覽、技能與信任、AI 設定、規則、Threat Cloud、威脅、指南
- WebSocket 即時推送狀態、事件與規則數量
- Token 驗證搭配自動開啟瀏覽器
- 深色主題搭配 Sage Green 色系，支援英文與繁體中文

### AI 平台自動設定

- `panguard setup` 一行指令自動偵測並配置 16 個 AI 平台
- **OpenClaw**：原生 Skill 整合至 `~/.openclaw/skills/panguard/SKILL.md`
- **QClaw**：MCP 伺服器註冊至 `~/.qclaw/mcp.json`
- Claude Code、Cursor、Codex CLI、Claude Desktop、Windsurf、WorkBuddy 透過 MCP
- 平台特定設定注入，零手動編輯

### 技能審計器為預設入口

- 8 項檢查安裝前安全閘門
- 偵測提示詞注入、工具投毒、秘密與不安全依賴
- 技能白名單：審計通過自動信任，偏移自動 revoke
- 整合至 Guard 儀表板（技能與信任頁面）

### 規則分發

- 安裝即附帶 71 ATR 偵測規則
- 每小時自動從 Threat Cloud 同步

### Threat Cloud LLM 審查

- Claude Sonnet 4 API 整合，自動化規則審查
- 社群投票管線處理提交的威脅情報
- 種子管線搭配 11 個威脅情報來源

### 一行安裝流程

- `curl -fsSL https://get.panguard.ai | bash` 觸發互動式設定
- 自動執行 `panguard setup` 配置 AI 平台
- 完成後自動啟動 Guard 與儀表板
- 瀏覽器自動開啟已驗證的儀表板頁面

## v0.7.0

**發佈日期：2025-02-25** -- 規則引擎增強與 osquery 整合

### 規則引擎增強

- 聚合表達式支援：`1 of them`、`all of them`、`1 of sel*`、`all of filter*`
- 新的匹配修飾器：`|cidr`（IP 範圍匹配）、`|gt`/`|gte`/`|lt`/`|lte`（數值比較）
- 新的編碼修飾器：`|base64`、`|base64offset`、`|utf8`、`|wide`
- 括號群組條件：`(sel_a OR sel_b) AND NOT filter`

### osquery 整合

- 新增 `OsqueryProvider`，支援以 SQL 查詢系統狀態
- 支援行程、連接埠、使用者、網路介面與核心模組查詢
- 未安裝 osquery 時自動降級至 shell 指令
