## v1.8.5 (2026-07-12)

**發佈日期：2026-07-12** -- ATR v3.5.8（747 條規則），OWASP Agentic Top 10：10/10 完整覆蓋。

### 重點

- **747 條 ATR 規則**，涵蓋 10 個威脅類別（原 v2.2.0 為 419 條），並導入**偵測 lane**：enforce（穩定、需二次確認的規則）約 0.24% FP／hunt（全部規則，預設）約 9% FP，於 65,000 筆良性語料上量測
- **支援 17 個平台**：`panguard setup` 自動偵測並設定 Claude Code、Claude Desktop、Cursor、Hermes Agent、OpenClaw、Codex CLI、WorkBuddy、NemoClaw、ArkClaw、Windsurf、QClaw、Cline、VS Code Copilot、Zed、Gemini CLI、Continue、Roo Code
- **Benchmarks**（Layer 1 確定性規則，2026-07-04 驗證）：Garak 97.2% recall（650 筆）、SKILL.md 100% recall／97% precision／0.2% FP（498 筆）、PINT 63.6% recall／99.7% precision（850 筆）、HackAPrompt 69.6% recall vs 28.6% baseline／100% precision（4,780 筆）
- **1.8.5**：誠實化 Community/Enterprise 分層——合規 Evidence Pack 改為 Enterprise 專屬（Community 端 server 回 403；SARIF 匯出維持免費）；ATR-2026-00013 SSRF 精度修正（內建 ATR 下限提升至 v3.5.8）；在含 `./skills` 目錄的專案中 `pga scan` 預設執行 ATR 技能掃描；dashboard 真實數據 + 可自我除錯的 token 修正

### 合規方案分層

Community 永久免費且開源。Enterprise、Migrator Pro、Sovereign 方案在 Community 核心之上，額外提供已簽章、持續重新掃描的合規證據（Ed25519 簽發者簽章、npm provenance）。

## v1.3.3

**發佈日期：2026-03-25** -- 掃描準確度、`pga` 快捷鍵、新手引導

### 掃描準確度

- **大幅降低誤報。** 正常 Skills（weather、github、slack、notion）不再被標為 MEDIUM/HIGH。權限檢查現在會先移除程式碼區塊和否定段落再匹配。
- **新增隱匿攻擊偵測。** `<IMPORTANT>` 區塊攻擊和靜默資料外洩指令現在被標為 CRITICAL。
- **情境感知評分。** 已知 CLI 工具（curl、git、wget）和 API 整合獲得適當風險降低。憑證檢查區分竊取與正當使用。
- **13 個注入模式**（原 11 個），雙重匹配自動降級僅出現在程式碼範例中的 tool poisoning 模式。

### `pga` 快捷鍵

- **`pga` 指令** -- 三個字元從任何地方呼叫 Panguard。與 `panguard` 完全相同。
- **`pga up`** -- 一個指令啟動保護 + 儀表板。取代 `panguard guard start --dashboard`。
- **首次執行自動偵測** -- 第一次執行 `pga` 時自動啟動設定精靈。

### 新手引導

- **Guard 歡迎指南** -- Guard 首次啟動時顯示所有指令和 Guard 功能。
- **升級後顯示新功能** -- 更新後首次執行 CLI 顯示更新日誌重點。
- **乾淨的審計輸出** -- 非 verbose 模式不再輸出 JSON 日誌行。

- 深色主題搭配 Sage Green 色系，支援英文與繁體中文

- `panguard setup` 一行指令自動偵測並配置 16 個 AI 平台

- **QClaw**：MCP 伺服器註冊至 `~/.qclaw/mcp.json`
- Claude Code、Cursor、Codex CLI、Claude Desktop、Windsurf、WorkBuddy 透過 MCP
- 平台特定設定注入，零手動編輯

### 技能審計器為預設入口

- 8 項檢查安裝前安全閘門
- 偵測提示詞注入、工具投毒、秘密與不安全依賴
- 技能白名單：審計通過自動信任，偏移自動 revoke
- 整合至 Guard 儀表板（技能與信任頁面）

### 規則分發

- 安裝即附帶 419 ATR 偵測規則
- 每小時自動從 Threat Cloud 同步

### Threat Cloud LLM 審查

- Claude Sonnet 4 API 整合，自動化規則審查
- 社群投票管線處理提交的威脅情報
- 種子管線搭配 11 個威脅情報來源

### 一行安裝流程

- `curl -fsSL https://get.panguard.ai | bash` 觸發互動式設定
- 自動執行 `panguard setup` 配置 AI 平台
- 完成後自動啟動 Guard 與儀表板
- 瀏覽器自動開啟已驗證的儀表板頁面
