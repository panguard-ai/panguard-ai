Panguard MCP 實作 [Model Context Protocol](https://modelcontextprotocol.io) 標準，讓 AI 助手可以用自然語言操作你的安全基礎設施。叫 AI 掃弱點、看 Guard 狀態、封 IP、產合規報告——全部用對話就搞定。

## 快速開始

把 Panguard 加到你 AI 助手的 MCP config：

<Tabs>
  <Tab title="Claude Desktop">
    編輯 `~/Library/Application Support/Claude/claude_desktop_config.json`（macOS）或 `%APPDATA%\Claude\claude_desktop_config.json`（Windows）：

    ```json
    {
      "mcpServers": {
        "panguard": {
          "command": "npx",
          "args": ["-y", "@panguard-ai/panguard-mcp"]
        }
      }
    }
    ```

    存檔後重開 Claude Desktop。

  </Tab>
  <Tab title="Cursor">
    加到 Cursor MCP 設定（Settings > MCP Servers）：

    ```json
    {
      "panguard": {
        "command": "npx",
        "args": ["-y", "@panguard-ai/panguard-mcp"]
      }
    }
    ```

  </Tab>
  <Tab title="Windsurf">

    ```json

        "panguard": {
          "command": "npx",
          "args": ["-y", "@panguard-ai/panguard-mcp"]

    }
    ```

  </Tab>
  <Tab title="Claude Code">

  </Tab>
</Tabs>

## 你可以做什麼

設定好之後，直接用自然語言跟 Panguard 互動：

| 你說的話                              | 對應的 MCP tool            |
| ------------------------------------- | -------------------------- |
| 「掃一下這台機器的弱點」              | `panguard_scan`            |
| 「幫我看 source code 有沒有安全問題」 | `panguard_scan_code`       |
| 「開啟即時防護」                      | `panguard_guard_start`     |
| 「停掉 Guard」                        | `panguard_guard_stop`      |
| 「現在安全狀態怎樣？」                | `panguard_status`          |
| 「最近有什麼警報？」                  | `panguard_alerts`          |
| 「封鎖 IP 203.0.113.50」              | `panguard_block_ip`        |
| 「產一份 PDF 安全報告」               | `panguard_generate_report` |
| 「初始化 Panguard config」            | `panguard_init`            |
| 「稽核這個 skill 目錄」               | `panguard_audit_skill`     |
| 「在這台機器上一鍵部署防護」          | `panguard_deploy`          |

## 11 個 Tool

| Tool                       | 類別   | 說明                           |
| -------------------------- | ------ | ------------------------------ |
| `panguard_scan`            | Scan   | 跑安全健檢（快速或完整）       |
| `panguard_scan_code`       | Scan   | 原始碼目錄的 SAST 掃描         |
| `panguard_guard_start`     | Guard  | 啟動即時監控 daemon            |
| `panguard_guard_stop`      | Guard  | 停止監控 daemon                |
| `panguard_status`          | System | 查所有 Panguard 服務狀態       |
| `panguard_alerts`          | Guard  | 查最近的安全警報，可按嚴重度篩 |
| `panguard_block_ip`        | Guard  | 手動封鎖 IP                    |
| `panguard_generate_report` | Report | 產 PDF 合規報告                |
| `panguard_init`            | System | 初始化 Panguard config         |
| `panguard_audit_skill`     | Scan   | 稽核 OpenClaw/AgentSkills 目錄 |
| `panguard_deploy`          | System | 一鍵部署：掃描 + Guard + 報告  |

## 架構

```
AI Assistant (Claude Desktop, Cursor, etc.)
         |
    MCP Protocol (stdio transport)
         |
   Panguard MCP Server (@panguard-ai/panguard-mcp)
         |
   ┌─────┼─────┬──────┬───────┐
   Scan  Guard  Report  System
```

MCP server 是跑在本機的 process，透過 stdio 跟 AI 助手溝通。所有安全操作都在你的機器上本地執行——沒有任何資料會透過 MCP 通道送到外部 server。

## 前置需求

- Node.js 18+（跑 `npx` 用）
- 裝好 Panguard CLI：`npm install -g @panguard-ai/panguard` 或直接 `npx panguard`
