這頁記錄 Panguard MCP server 提供的每個工具。每個工具都列出說明、參數 schema 跟用法範例。

---

## panguard_scan

對本機跑一次安全健檢。回傳風險分數（0--100）、等級（A--F）、安全發現項目清單。

<ParamField path="depth" type="string" default="quick">
  掃描深度：`quick`（約 30 秒）或 `full`（約 60 秒）。
</ParamField>

<ParamField path="lang" type="string" default="en">
  輸出語言：`en` 或 `zh-TW`。
</ParamField>

**範例提示：**「用中文對這台機器跑完整安全掃描。」

**範例回應：** 風險分數 72/100（等級 C），8 項發現（1 critical、2 high、3 medium、2 low）。

---

## panguard_scan_code

掃描原始碼目錄找安全漏洞（SAST）。偵測 SQL injection、XSS、寫死的密鑰、command injection 等。

<ParamField path="dir" type="string" required>
  要掃的原始碼目錄。絕對或相對路徑。
</ParamField>

<ParamField path="lang" type="string" default="en">
  輸出語言：`en` 或 `zh-TW`。
</ParamField>

**範例提示：**「檢查 ./src 目錄有沒有安全漏洞。」

---

## panguard_guard_start

啟動 Panguard Guard 即時威脅監控 daemon。

<ParamField path="dataDir" type="string">
  資料目錄路徑。預設 `~/.panguard-guard`。
</ParamField>

<ParamField path="mode" type="string">
  運作模式：`learning` 或 `protection`。
</ParamField>

**範例提示：**「用學習模式啟動即時防護。」

---

## panguard_guard_stop

停止 Panguard Guard daemon。

<ParamField path="dataDir" type="string">
  資料目錄路徑。預設 `~/.panguard-guard`。
</ParamField>

**範例提示：**「停止 Guard。」

---

## panguard_status

取得所有 Panguard 服務（Guard、Scan、Manager）的目前狀態。回傳執行狀態、威脅數量、系統資訊。

<ParamField path="dataDir" type="string">
  資料目錄路徑。預設 `~/.panguard-guard`。
</ParamField>

**範例提示：**「這台機器現在的安全狀態怎樣？」

**回傳：** Guard 狀態、模式、uptime、安全分數、活躍威脅、處理過的事件數。

---

## panguard_alerts

拿 Panguard Guard 偵測到的最近安全警示。回傳最新的威脅事件，附嚴重性跟詳細資訊。

<ParamField path="limit" type="number" default="20">
  最多回傳幾筆警示。
</ParamField>

<ParamField path="severity" type="string" default="all">
  依嚴重性篩選：`critical`、`high`、`medium`、`low` 或 `all`。
</ParamField>

<ParamField path="dataDir" type="string">
  資料目錄路徑。預設 `~/.panguard-guard`。
</ParamField>

**範例提示：**「給我最近 5 個 critical 等級的警示。」

---

## panguard_block_ip

手動封鎖一個 IP，禁止它存取系統。

<ParamField path="ip" type="string" required>
  要封鎖的 IP（IPv4 或 IPv6）。
</ParamField>

<ParamField path="duration" type="string" default="1h">
  封鎖時間。例如：`1h`、`24h`、`permanent`。
</ParamField>

<ParamField path="reason" type="string">
  封鎖原因（記在 action manifest 裡）。
</ParamField>

**範例提示：**「封鎖 IP 203.0.113.50 24 小時，因為它在做 port scan。」

---

## panguard_generate_report

從掃描結果產生 PDF 合規報告。回傳產生的 PDF 路徑。

<ParamField path="output" type="string" default="./panguard-report.pdf">
  輸出 PDF 路徑。
</ParamField>

<ParamField path="lang" type="string" default="en">
  報告語言：`en` 或 `zh-TW`。
</ParamField>

<ParamField path="depth" type="string" default="full">
  底層掃描的深度：`quick` 或 `full`。
</ParamField>

**範例提示：**「產生繁體中文的 PDF 安全報告。」

---

## panguard_init

初始化 Panguard config，用預設值（非互動模式）。

<ParamField path="dataDir" type="string">
  資料目錄路徑。預設 `~/.panguard-guard`。
</ParamField>

<ParamField path="lang" type="string" default="en">
  預設語言：`en` 或 `zh-TW`。
</ParamField>

<ParamField path="mode" type="string" default="learning">
  初始運作模式：`learning` 或 `protection`。
</ParamField>

**範例提示：**「用中文預設值初始化 Panguard。」

---

## panguard_audit_skill

稽核 OpenClaw/AgentSkills 的 `SKILL.md` 目錄找安全問題。檢查 manifest 有效性、prompt injection、tool poisoning、程式碼漏洞、相依套件、權限。

<ParamField path="path" type="string" required>
  含 `SKILL.md` 的 skill 目錄路徑。
</ParamField>

**回傳：** 風險分數（0--100）加上涵蓋以下面向的詳細發現：
- Manifest 有效性
- Prompt injection 偵測
- Tool poisoning 分析
- 程式碼漏洞掃描
- 相依套件稽核
- 權限審查

**範例提示：**「稽核 ./my-agent-skill 的安全問題。」

---

## panguard_deploy

一鍵部署 Panguard 服務：掃描弱點、啟動 Guard 監控、產生初始報告。

<ParamField path="dataDir" type="string">
  資料目錄路徑。預設 `~/.panguard-guard`。
</ParamField>

<ParamField path="lang" type="string" default="en">
  語言：`en` 或 `zh-TW`。
</ParamField>

<ParamField path="mode" type="string" default="learning">
  初始 Guard 模式：`learning` 或 `protection`。
</ParamField>

<ParamField path="generateReport" type="boolean" default="true">
  掃完要不要產生 PDF 報告。
</ParamField>

**範例提示：**「在這台機器上部署完整防護，順便產 PDF 報告。」

**執行：** `panguard scan` + `panguard guard start` + `panguard scan --output report.pdf`

## 工具開放性

所有工具都免費可用。Panguard 是 MIT 授權的 100% 開源專案。

<Tip>
  所有工具回傳結構化 JSON，AI 助手可以 parse 後用自然語言呈現。回應裡的 `isError` 欄位代表操作是成功還是失敗。
</Tip>
