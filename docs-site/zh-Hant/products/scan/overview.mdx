Panguard Scan 是一套快速、全面的安全稽核工具，60 秒內就能評估系統的安全狀態。跑 10 種專用掃描器，產出 0--100 風險分數配 A--F 等級，還能選擇性產出專業 PDF 報告。

## 快速開始

```bash
# 快速掃描（約 30 秒）
panguard scan --quick

# 完整掃描（約 60 秒）
panguard scan

# 產出 PDF 報告
panguard scan --output report.pdf

# 繁體中文輸出
panguard scan --lang zh-TW
```

## 嚴重等級

每個弱點都會被分到五個等級之一：

| 等級         | 意義                 | 處理時程    |
| ------------ | -------------------- | ----------- |
| **CRITICAL** | 有立即被入侵的風險   | 馬上修      |
| **HIGH**     | 嚴重安全風險         | 24 小時內修 |
| **MEDIUM**   | 中等風險，建議改善   | 一週內修    |
| **LOW**      | 低風險，最佳實務建議 | 有空再修    |
| **INFO**     | 純資訊               | 看過就好    |

## 產品搭配

### Scan + Guard

先用 Scan 建 baseline，再開 Guard 持續監控：

```bash
# 1. 跑初始掃描
panguard scan --output baseline.pdf

# 2. 修掃出來的問題

# 3. 啟動持續防護
panguard guard start

# 4. 定期再掃一次追蹤改善
panguard scan --output weekly-check.pdf
```

### Scan + Report

掃描結果直接餵進合規報告：

```bash
panguard scan --output scan-result.pdf
```

## CLI 選項

```
panguard scan [options]

Options:
  --quick              快速模式（約 30 秒）
  --output <path>      PDF 報告輸出路徑
  --lang <en|zh-TW>    語言（預設：en）
  --verbose            詳細輸出
  --json               純 JSON 輸出（給 AI agent 用）
  --target <host>      遠端目標（IP 或 domain）
```
