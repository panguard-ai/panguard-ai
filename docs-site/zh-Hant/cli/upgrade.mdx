description: '把 Panguard CLI 更新到最新版。'

---

`upgrade` 檢查有沒有新版本，然後自動下載安裝。binary 替換、資料庫 migration、升級後驗證一條龍搞定。

## 用法

```bash
panguard upgrade
```

## 升級流程

1. **比對版本** -- 查你裝的版本跟最新 release
2. **下載新 binary** -- 從官方 release channel 下載
3. **驗證 checksum** -- 確保檔案完整性
4. **替換 binary** -- 原地替換
5. **跑 migration** -- 如果資料格式有變的話
6. **驗證升級** -- 跑 self-test 確認沒問題
