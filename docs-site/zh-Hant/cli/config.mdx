`config` 打開目前 Panguard config 的互動式介面，讓你檢查跟修改所有服務的設定 -- Guard、Scan 等等。

## 用法

```bash
panguard config
```

## 顯示內容

跑 `panguard config` 會列出 `~/.panguard/config.yaml` 裡面的現行設定，包括：

- **General** -- 語言、資料目錄、Threat Cloud endpoint
- **Guard** -- 開了哪些監控、自動回應規則、學習期
- **Scan** -- 掃描器模組、輸出偏好、排程
- **通知** -- 通知頻道、嚴重性門檻、靜音時段
- **Report** -- 預設框架、輸出格式、組織名稱

## 範例

<CodeGroup>
```bash 看目前 config
panguard config
```
</CodeGroup>

<Tip>
  要直接改檔案的話，config 放在 `~/.panguard/config.yaml`。隨便用什麼編輯器改 -- 下次跑 Panguard 指令或 Guard 重啟時就會吃到新設定。
</Tip>

## Config 檔位置

| 平台 | 預設路徑 |
| --- | --- |
| Linux | `~/.panguard/config.yaml` |
| macOS | `~/.panguard/config.yaml` |
| Windows (WSL) | `~/.panguard/config.yaml` |
