`scan` 會對你的端點做一次完整的安全評估：弱點、錯誤設定、過時的套件、已知威脅通通掃一遍。結果會透過 AI 排序跟評分，讓你先處理最嚴重的問題。

## 用法

```bash
panguard scan [options]
```

## 選項

<ParamField path="--quick" type="flag">
  快速掃描，只跑關鍵項目。
</ParamField>

<ParamField path="--output" type="string">
  把掃描結果寫到指定路徑。根據副檔名自動判斷格式，支援 `.json`、`.html`、`.pdf`。
</ParamField>

<ParamField path="--lang" type="string" default="en">
  設定掃描結果的輸出語言（例如 `en`、`zh-TW`、`ja`）。
</ParamField>

<ParamField path="--verbose" type="flag">
  顯示詳細輸出，包含每項檢查的進度跟 debug 資訊。
</ParamField>

## 範例

<CodeGroup>
```bash 快速掃描
panguard scan --quick
```

```bash 完整掃描，輸出 JSON
panguard scan --output ./results/scan-report.json
```

```bash 繁體中文完整掃描
panguard scan --lang zh-TW --verbose
```

```bash 匯出 PDF 報告
panguard scan --output ~/Desktop/security-audit.pdf
```

</CodeGroup>

## 掃描模式

| 模式         | 檢查項目                                                | 時間     |
| ------------ | ------------------------------------------------------- | -------- |
| `--quick`    | 關鍵弱點、開放 port、root 存取                          | 約 30 秒 |
| 完整（預設） | 全部掃描器，含 CVE、CIS benchmark、惡意軟體特徵碼、網路 | 2-5 分鐘 |

## 掃什麼

- **系統弱點** -- 比對已安裝套件的 CVE 資料庫
- **設定稽核** -- CIS benchmark 作業系統強化檢查
- **網路暴露** -- 開放 port、listening service、防火牆規則
- **惡意軟體偵測** -- ATR 規則比對跟 hash 查詢
- **檔案完整性** -- 跟已知 baseline 比對
- **使用者帳號** -- 權限提升風險、弱密碼
