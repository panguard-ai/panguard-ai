## 一行搞定

<Tabs>
  <Tab title="macOS / Linux">```bash curl -fsSL https://get.panguard.ai | bash ```</Tab>
  <Tab title="Windows">```powershell irm https://get.panguard.ai/windows | iex ```</Tab>
  <Tab title="npm（全平台）">
    ```bash npm install -g @panguard-ai/panguard && pga setup && pga guard start
    --dashboard ```
  </Tab>
</Tabs>

一行指令做完所有事：安裝 Panguard、連接你的 AI agent（Claude Code、Cursor、OpenClaw 等）、啟動 Guard、Dashboard 自動在 `http://127.0.0.1:9100` 開啟。

---

## 掃描你的 AI Skill

裝好之後，審計你系統上的 MCP skill：

```bash
panguard audit skill .
```

```
Risk Score: 8/100 (LOW)
[PASS] Manifest: valid SKILL.md
[PASS] Prompt Safety: no injection patterns
[PASS] Secrets: none found
[PASS] Code: no suspicious patterns

VERDICT: SAFE TO INSTALL
```

掃描所有平台上已安裝的 skill：

```bash
panguard guard scan
```

用 61 條 ATR 規則檢查所有 skill — prompt injection、tool poisoning、credential theft 等 9 大威脅類別。

<Warning>
  **CRITICAL 和 HIGH 的發現需要你處理。** 逐一檢查每個發現，決定要保留、更新還是移除。
</Warning>

## 看 Guard 狀態

```bash
panguard guard status
```

```
Status:     RUNNING
Mode:       learning (Day 1/7)
Rules:      61 ATR detection rules
Dashboard:  http://127.0.0.1:9100
```

<Info>
  **學習模式（第 1-7 天）：** Guard 觀察你的正常行為並建立 baseline。這段期間不會誤報。第 7
  天之後異常偵測自動啟動。
</Info>

## 飛輪怎麼運作

跑 Panguard 的同時，你的機器就加入了集體防禦網路：

1. **Guard 在你的機器上偵測到威脅。**
2. **匿名 hash 分享到 Threat Cloud**（不含個資、不含原始碼）。
3. **3 個獨立掃描器確認**同樣的 pattern — 自動產生新 ATR 規則。
4. **所有 Panguard 使用者在 1 小時內收到新規則。**

一台機器被攻擊，一小時後所有機器都免疫。

## 常用指令

| 指令                               | 功能                              |
| ---------------------------------- | --------------------------------- |
| `panguard setup`                   | 自動偵測並連接 AI 平台            |
| `panguard audit skill <path>`      | 安裝前審計單一 skill              |
| `panguard guard scan`              | 掃描所有平台上已安裝的 skill      |
| `panguard guard start --dashboard` | 啟動 24/7 監控 + 瀏覽器 Dashboard |
| `panguard guard stop`              | 停止 Guard                        |
| `panguard scan --quick`            | 系統安全掃描（port、服務）        |
