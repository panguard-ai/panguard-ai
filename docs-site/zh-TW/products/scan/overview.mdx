Panguard Scan 是一款快速、全面的安全稽核工具，能在 60 秒內評估您系統的安全態勢。它執行 10 個專業掃描器，產生 0--100 的風險評分搭配 A--F 字母等級，並可選擇性地產生專業 PDF 報告。

## 快速開始

```bash
# 快速掃描（約 30 秒）
panguard scan --quick

# 完整掃描（約 60 秒）
panguard scan

# 產生 PDF 報告
panguard scan --output report.pdf

# 繁體中文輸出
panguard scan --lang zh-TW
```

## 嚴重等級

每個發現項目被分類為五個嚴重等級之一：

| 等級 | 含義 | 處理時程 |
|-------|---------|-----------------|
| **CRITICAL** | 立即受損風險 | 立即修復 |
| **HIGH** | 重大安全風險 | 24 小時內修復 |
| **MEDIUM** | 中等風險，建議改善 | 一週內修復 |
| **LOW** | 低風險，最佳實務建議 | 方便時修復 |
| **INFO** | 資訊性發現 | 確認即可 |

## 產品搭配

### Scan + Guard

使用 Scan 建立基準線，然後啟動 Guard 進行持續監控：

```bash
# 1. 執行初始掃描
panguard scan --output baseline.pdf

# 2. 修復發現的問題

# 3. 開始持續防護
panguard guard start

# 4. 定期重新掃描以追蹤改善情況
panguard scan --output weekly-check.pdf
```

### Scan + Report

掃描發現項目直接導入合規報告產生：

```bash
panguard scan --output scan-result.pdf
```

## CLI 選項

```
panguard scan [options]

Options:
  --quick              快速模式（約 30 秒）
  --output <path>      PDF 報告輸出路徑
  --lang <en|zh-TW>    語言（預設：en）
  --verbose            詳細輸出
  --json               純 JSON 輸出（供 AI 代理使用）
  --target <host>      遠端目標（IP 或網域）
```
