# 首次掃描

Panguard Scan 約 60 秒內分析您的系統、識別安全弱點，並提供可操作的修復指導。快速掃描免費；完整掃描需 Solo 方案或以上。

---

## 前置條件

執行首次掃描前，請確認已安裝 Panguard：

```bash
# 透過 npm 安裝
npm install -g @panguard-ai/panguard

# 或透過一行安裝程式
curl -fsSL https://get.panguard.ai | bash

# 驗證安裝
panguard --version
```

## 執行首次掃描

<Steps>
  <Step title="執行快速掃描">
    先用快速掃描取得即時概覽（免費，無需登入）：

    ```bash
    panguard scan --quick
    ```

    約 30 秒完成，涵蓋作業系統偵測、網路介面、開放連接埠、執行中服務、密碼策略、防火牆狀態與安全工具偵測。
  </Step>
  <Step title="查看結果">

## 快速掃描 vs. 完整掃描

| 功能 | 快速模式 (`--quick`) | 完整模式（預設） |
| --- | --- | --- |
| **持續時間** | ~30 秒 | ~60 秒 |
| **所需方案** | 免費 | Solo |
| **作業系統偵測** | 是 | 是 |
| **網路介面** | 是 | 是 |
| **開放連接埠** | 是 | 是 |
| **執行中服務** | 是 | 是 |
| **密碼策略** | 是 | 是 |
| **防火牆狀態** | 是 | 是 |
| **安全工具** | 是 | 是 |
| **SSL 憑證驗證** | 否 | 是 |
| **排程任務稽核** | 否 | 是 |
| **共享資料夾安全** | 否 | 是 |
| **風險分數** | 是 | 是 |

執行完整掃描：

```bash
# 先登入（Solo 方案必要）
panguard login

# 執行完整掃描
panguard scan
```

## 嚴重性等級

| 嚴重性 | 意義 | 建議動作 |
| --- | --- | --- |
| **CRITICAL** | 有立即被入侵的風險 | 立即修復 |
| **HIGH** | 重大安全風險 | 24 小時內修復 |
| **MEDIUM** | 中度風險，建議改善 | 一週內修復 |
| **LOW** | 低風險，最佳實務建議 | 方便時修復 |
| **INFO** | 資訊性發現 | 無需行動 |

## 產生 PDF 報告

將掃描結果匯出為專業 PDF 報告：

```bash
# 英文報告
panguard scan --output my-report.pdf

# 繁體中文報告
panguard scan --output my-report.pdf --lang zh-TW
```

PDF 包含：
1. **封面** -- 組織名稱、掃描日期、品牌標識
2. **執行摘要** -- 風險分數、等級、發現統計
3. **發現詳情** -- 每項發現的嚴重性、說明與位置
4. **修復指導** -- 每項發現的具體修復步驟
5. **合規對應** -- 發現對應至 ISO 27001 / SOC 2 / 台灣資安法
