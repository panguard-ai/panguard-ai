## 開始使用

Panguard 是 100% 開源專案（MIT 授權）。無需帳號、無需信用卡，所有功能完全免費。

<Steps>
  <Step title="安裝 CLI">
    ```bash
    curl -fsSL https://get.panguard.ai | bash
    ```

    或使用 npm：

    ```bash
    npm install -g @panguard-ai/panguard
    ```
  </Step>

  <Step title="配置 AI Agent">
    一行指令自動配置所有偵測到的 AI 平台：

    ```bash
    panguard setup
    ```

    自動偵測並配置 Claude Code、Cursor、OpenClaw、Codex CLI、Claude Desktop 和 Windsurf。
  </Step>

  <Step title="執行首次掃描">
    ```bash
    panguard scan --quick
    ```
  </Step>
</Steps>
