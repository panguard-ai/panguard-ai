## 建立帳號

<Steps>
  <Step title="在網站註冊">
    前往 [panguard.ai](https://panguard.ai)，點擊右上角的**註冊**。

    您可以使用以下方式註冊：
    - **Google 帳號** -- 一鍵 OAuth 登入
    - **Email + 密碼** -- 傳統註冊方式，需進行 Email 驗證
  </Step>

  <Step title="選擇方案">
    選擇適合您需求的方案。Community 方案永久免費，無需信用卡。

    | 方案 | 價格 | 設備數 | 重點功能 |
    | --- | --- | --- | --- |
    | **Community** | 免費 | 1 | Scan + Layer 1 Guard（僅規則） |
    | **Solo** | $9/月 | 3 | 完整三層堆疊、基礎 AI |
    | **Pro** | $29/月 | 10 | Report、Trap、Chat、雲端 AI |
    | **Business** | $79/月 | 25 | 自訂 AI、API 存取、90+ 天日誌 |

    <Info>
      所有付費方案皆提供年繳 8 折優惠。前往 [panguard.ai/pricing](https://panguard.ai/pricing) 詳細比較各方案。
    </Info>
  </Step>

  <Step title="確認帳號">
    如果您使用 Email 註冊，請檢查收件匣中的驗證信並確認帳號。Google OAuth 使用者會自動完成驗證。
  </Step>
</Steps>

## CLI 驗證

建立帳號後，驗證 CLI 以連結您的訂閱。

### 標準登入

```bash
panguard login
```

CLI 會自動開啟您的預設瀏覽器。完成驗證流程後，返回終端機：

```
  PANGUARD AI

  -- Login Info ----------------------------------------

  Email     user@example.com
  Name      User Name
  Plan      Solo
  Expires   2026-04-27

  Login successful!
```

### 驗證連線狀態

```bash
panguard whoami
```

```
  PANGUARD AI

  -- Account Info ----------------------------------------

  Email     user@example.com
  Name      User Name
  Plan      Solo
  Expires   2026-04-27
  Server    https://panguard.ai
```

<Info>
  CLI 連線有效期為 30 天。過期後請再次執行 `panguard login` 重新驗證。
</Info>

### 無圖形介面與 SSH 伺服器

如果您在沒有瀏覽器的伺服器上作業：

```bash
panguard login --no-browser
```

這會印出一個驗證 URL，您可以複製並在任何有瀏覽器的裝置上開啟。在該裝置上完成驗證後，CLI 連線會自動啟用。

```
  Open the following URL in your browser to complete login:
  https://panguard.ai/login?cli_state=abc123&callback=...

  Waiting for authentication...
```

<Tip>
  `--no-browser` 旗標適用於 CI/CD 流水線、Docker 容器與透過 SSH 存取的遠端伺服器。
</Tip>

## 訂閱管理

### 升級方案

您可以隨時透過網站或 CLI 升級：

```bash
panguard upgrade
```

或直接前往 [panguard.ai/pricing](https://panguard.ai/pricing)。

### 查看目前方案

```bash
panguard whoami
```

### 登出

```bash
panguard logout
```

這會移除本地的驗證 Token。您的帳號與訂閱仍然有效。

## 疑難排解

<AccordionGroup>
  <Accordion title="瀏覽器未自動開啟">
    使用 `--no-browser` 旗標並手動開啟印出的 URL：

    ```bash
    panguard login --no-browser
    ```
  </Accordion>

  <Accordion title="功能被方案層級限制">
    如果您看到訊息表示您的方案不支援某項功能，請前往 [panguard.ai/pricing](https://panguard.ai/pricing) 升級您的訂閱。
  </Accordion>

  <Accordion title="Token 已過期">
    CLI 連線 30 天後過期。請重新驗證：

    ```bash
    panguard login
    ```
  </Accordion>
</AccordionGroup>
