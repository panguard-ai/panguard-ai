## GET /api/query/timeseries

回傳威脅活動的時間序列資料，適用於圖表繪製。

<ParamField query="granularity" type="string" default="day">
  時間區間大小：`hour`、`day` 或 `week`。
</ParamField>

<ParamField query="period" type="string" default="30d">
  時間範圍：`24h`、`7d`、`30d`、`90d`。
</ParamField>

<ParamField query="type" type="string">
  依 IoC 類型篩選。
</ParamField>

<ParamField query="category" type="string">
  依威脅類別篩選。
</ParamField>

```bash cURL
curl -X GET "https://tc.panguard.ai/api/query/timeseries?granularity=day&period=7d" \
```

## POST /api/sightings

記錄現有 IoC 的新觀測。觀測會增加指標的聲譽分數權重。

<ParamField body="value" type="string" required>
  被觀測到的 IoC 值。
</ParamField>

<ParamField body="source" type="string" required>
  觀測來源（例如 `guard-agent`、`honeypot`）。
</ParamField>

<ParamField body="context" type="object">
  關於觀測的額外上下文。
</ParamField>

<CodeGroup>
```bash cURL
curl -X POST "https://tc.panguard.ai/api/sightings" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "value": "198.51.100.42",
    "source": "guard-agent",
    "context": {
      "agentId": "agent-a1b2c3d4",
      "detectionType": "brute_force",
      "targetPort": 22
    }
  }'
```
</CodeGroup>

<ResponseExample>
```json 201
{
  "ok": true,
  "data": {
    "sightingId": "sig_x1y2z3",
    "iocValue": "198.51.100.42",
    "totalSightings": 48,
    "reputationChange": -2
  }
}
```
</ResponseExample>

## GET /api/audit-log

回傳針對 Threat Cloud 執行的 API 操作稽核日誌。用於合規和除錯。

<ParamField query="action" type="string">
  依操作類型篩選：`create`、`update`、`delete`、`query`。
</ParamField>

<ParamField query="since" type="string">
  日誌起始的 ISO 8601 時間戳。
</ParamField>

<ParamField query="limit" type="number" default="50">
  最大結果數（最大 200）。
</ParamField>

<CodeGroup>
```bash cURL
curl -X GET "https://tc.panguard.ai/api/audit-log?action=create&limit=10" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</CodeGroup>

<ResponseExample>
```json 200
{
  "ok": true,
  "data": [
    {
      "id": "audit_001",
      "action": "create",
      "resource": "ioc",
      "details": "Added 3 IoCs (2 new, 1 updated)",
      "apiKey": "key_...redacted",
      "ip": "203.0.113.10",
      "timestamp": "2026-03-07T08:15:00.000Z"
    },
    {
      "id": "audit_002",
      "action": "create",
      "resource": "sighting",
      "details": "Recorded sighting for 198.51.100.42",
      "apiKey": "key_...redacted",
      "ip": "203.0.113.10",

    }
  ],
  "pagination": {
    "total": 1234,

}
```
</ResponseExample>

</Info>
