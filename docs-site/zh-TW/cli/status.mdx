`status` 讓你一目了然 Panguard 安裝的全貌 -- Guard daemon 健不健康、上次掃描結果、通知頻道連線狀態等等。

## 用法

```bash
panguard status [options]
```

## 選項

<ParamField path="--json" type="flag">
  輸出結構化 JSON，方便腳本或自動化使用。
</ParamField>

<ParamField path="--config" type="string">
  自訂 config 檔路徑。預設 `~/.panguard/config.yaml`。
</ParamField>

<ParamField path="--lang" type="string" default="en">
  設定輸出語言。
</ParamField>

## 範例

<CodeGroup>
```bash 看系統狀態
panguard status
```

```bash JSON 輸出，給腳本用
panguard status --json
```

```bash 指定 config 看狀態
panguard status --config /etc/panguard/config.yaml
```
</CodeGroup>

## 輸出區塊

status 指令回報的區塊：

| 區塊 | 資訊 |
| --- | --- |
| **Guard** | 跑著 / 停了、uptime、處理過的事件數 |
| **Scan** | 上次掃描時間、發現數、風險分數 |
| **通知頻道** | 已設定的頻道、連線健康狀態 |
| **System** | Panguard 版本、OS、config 路徑 |
