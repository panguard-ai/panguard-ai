---
title: "panguard status"
description: "顯示所有 Panguard 服務的整體健康狀態。"
icon: "circle-info"
---

`status` 指令提供 Panguard 安裝的統一檢視 -- Guard 代理健康、最近掃描結果、活躍蜜罐、通知頻道連線等。

## 使用方式

```bash
panguard status [options]
```

## 選項

<ParamField path="--json" type="flag">
  以結構化 JSON 輸出狀態資訊，用於腳本和自動化。
</ParamField>

<ParamField path="--config" type="string">
  自訂設定檔的路徑。預設為 `~/.panguard/config.yaml`。
</ParamField>

<ParamField path="--lang" type="string" default="en">
  設定輸出語言。
</ParamField>

## 範例

<CodeGroup>
```bash 顯示系統狀態
panguard status
```

```bash 用於腳本的 JSON 輸出
panguard status --json
```

```bash 使用自訂設定的狀態
panguard status --config /etc/panguard/config.yaml
```
</CodeGroup>

## 輸出區段

status 指令回報以下區段：

| 區段 | 資訊 |
| --- | --- |
| **Guard** | 運行中 / 已停止、運行時間、已處理事件 |
| **Scan** | 上次掃描時間、發現數量、風險評分 |
| **Chat** | 已設定頻道、連線健康狀態 |
| **Trap** | 活躍蜜罐、已擷取互動 |
| **System** | Panguard 版本、作業系統、設定路徑 |
