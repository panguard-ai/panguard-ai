---
title: "panguard logout"
description: "結束目前的 Panguard CLI 工作階段並清除已儲存的憑證。"
icon: "right-from-bracket"
---

`logout` 指令透過移除已儲存的存取 token 來結束您已認證的工作階段。登出後，需要身份驗證的指令會提示您重新登入。

## 使用方式

```bash
panguard logout [options]
```

## 選項

<ParamField path="--lang" type="string" default="en">
  設定登出訊息的語言。
</ParamField>

## 範例

<CodeGroup>
```bash 登出
panguard logout
```

```bash 以中文訊息登出
panguard logout --lang zh-TW
```
</CodeGroup>

## 執行內容

1. **使本機 token 失效**，儲存於 `~/.panguard/credentials`
2. **通知伺服器**撤銷工作階段（如果可連線）
3. **清除快取的使用者資料**從本機設定中
