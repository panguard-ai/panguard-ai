`scan` 指令對您的端點執行全面的安全評估，檢查弱點、錯誤設定、過時軟體和已知威脅。結果由 AI 評分和排序，讓您專注於最重要的問題。

## 使用方式

```bash
panguard scan [options]
```

## 選項

<ParamField path="--quick" type="flag">
  執行僅涵蓋關鍵檢查的快速掃描。適用於所有方案，包括 Community（免費）。
</ParamField>

<ParamField path="--output" type="string">
  將掃描結果寫入指定檔案路徑。根據副檔名支援 `.json`、`.html` 和 `.pdf` 格式。
</ParamField>

<ParamField path="--lang" type="string" default="en">
  設定掃描結果的輸出語言（例如 `en`、`zh-TW`、`ja`）。
</ParamField>

<ParamField path="--verbose" type="flag">
  顯示詳細輸出，包括個別檢查進度和除錯資訊。
</ParamField>

## 範例

<CodeGroup>
```bash 快速掃描（免費方案）
panguard scan --quick
```

```bash 完整掃描並輸出 JSON
panguard scan --output ./results/scan-report.json
```

```bash 以繁體中文進行完整掃描
panguard scan --lang zh-TW --verbose
```

```bash 匯出 PDF 報告
panguard scan --output ~/Desktop/security-audit.pdf
```
</CodeGroup>

## 掃描類型

| 模式 | 檢查項目 | 持續時間 | 所需方案 |
| --- | --- | --- | --- |
| `--quick` | 關鍵弱點、開放埠、root 存取 | 約 30 秒 | Community（免費） |
| 完整（預設） | 所有掃描器，包括 CVE、CIS 基準、惡意軟體特徵、網路 | 2-5 分鐘 | Solo 以上 |

## 掃描內容

- **系統弱點** -- 根據已安裝套件進行 CVE 資料庫比對
- **設定稽核** -- CIS 基準檢查，用於作業系統強化
- **網路暴露** -- 開放埠、監聽服務、防火牆規則
- **惡意軟體偵測** -- YARA 規則比對和雜湊值查詢
- **檔案完整性** -- 與已知良好基線進行比對
- **使用者帳號** -- 權限提升風險、弱密碼
