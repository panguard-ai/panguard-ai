Panguard CLI 是您進行掃描、防護、監控及管理端點安全的統一入口。從快速弱點掃描到部署蜜罐網路，所有功能都可透過一個二進位檔案存取。

## 使用方式

```bash
panguard <command> [options]
```

## 全域選項

| 選項 | 說明 |
| --- | --- |
| `--help` | 顯示任何指令的說明 |
| `--version` | 輸出已安裝的 Panguard CLI 版本 |
| `--lang <lang>` | 設定輸出語言（`en`、`zh-TW`、`ja` 等） |

## 指令分類

### 掃描

| 指令 | 說明 |
| --- | --- |
| [`scan`](/cli/scan) | 在本機執行安全掃描 |

### 防護

| 指令 | 說明 |
| --- | --- |
| [`guard`](/cli/guard) | 即時端點防護代理 |
| [`trap`](/cli/trap) | 蜜罐 / 欺敵技術 |
| [`chat`](/cli/chat) | 通知頻道管理（Telegram、Slack、Email） |
| [`report`](/cli/report) | 合規報告產生 |

### 系統

| 指令 | 說明 |
| --- | --- |
| [`init`](/cli/init) | 初始化當前機器的設定 |
| [`deploy`](/cli/deploy) | 將設定部署到遠端端點 |
| [`status`](/cli/status) | 顯示整體系統健康狀態 |
| [`config`](/cli/config) | 檢視或編輯設定 |
| [`doctor`](/cli/doctor) | 診斷常見系統問題 |
| [`upgrade`](/cli/upgrade) | 將 Panguard CLI 更新至最新版本 |

### 身份驗證

| 指令 | 說明 |
| --- | --- |
| [`login`](/cli/login) | 使用 Panguard 帳號進行身份驗證 |
| [`logout`](/cli/logout) | 結束目前的工作階段 |
| [`whoami`](/cli/whoami) | 顯示目前已驗證的使用者 |

### 進階

| 指令 | 說明 |
| --- | --- |
| [`manager`](/cli/manager) | 啟動多端點 Manager 伺服器 |
| [`threat`](/cli/threat) | 啟動本機 Threat Cloud 情報伺服器 |

## 安裝

如果您尚未安裝 CLI，請執行：

```bash
curl -fsSL https://get.panguard.ai | bash
```

<Info>
  請參閱[安裝](/installation)頁面，瞭解包括 Homebrew、APT 和 Docker 在內的各平台安裝說明。
