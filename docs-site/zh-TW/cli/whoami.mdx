`whoami` 指令顯示目前登入使用者的資訊，包括 email、方案層級、組織和 token 到期時間。

## 使用方式

```bash
panguard whoami [options]
```

## 選項

<ParamField path="--json" type="flag">
  以結構化 JSON 輸出使用者資訊，用於腳本和自動化。
</ParamField>

<ParamField path="--lang" type="string" default="en">
  設定輸出語言。
</ParamField>

## 範例

<CodeGroup>
```bash 顯示目前使用者
panguard whoami
```

```bash 用於腳本的 JSON 輸出
panguard whoami --json
```
</CodeGroup>

## 範例輸出

<Tabs>
  <Tab title="人類可讀">
    ```
    Email:   user@example.com
    License: MIT (Open Source)
    Org:     Acme Corp
    Version: 0.3.3
    ```
  </Tab>
  <Tab title="JSON (--json)">
    ```json
    {
      "email": "user@example.com",
      "plan": "pro",
      "org": "Acme Corp",
      "machines_used": 4,
      "machines_limit": 10,
      "token_expires_at": "2026-04-04T12:00:00Z"
    }
    ```
  </Tab>
</Tabs>
