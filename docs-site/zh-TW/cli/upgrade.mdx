---

`upgrade` 指令檢查並安裝最新版本的 Panguard CLI。它會自動偵測目前安裝方式，下載最新版本並完成升級。

## 使用方式

```bash
panguard upgrade
```

## 執行內容

1. **檢查目前版本**與 npm registry 上最新版本的差異
2. **下載最新版本**透過 npm
3. **驗證安裝**確保新版本正確運行
4. **執行遷移**（如果需要資料格式變更）

## 替代升級方式

如果您使用 npm 全域安裝，也可以直接透過 npm 更新：

```bash
npm update -g @panguard-ai/panguard
```
