---
title: "panguard upgrade"
description: "將 Panguard CLI 更新至最新版本。"
icon: "arrow-up"
---

`upgrade` 指令檢查並安裝最新版本的 Panguard CLI。它會自動處理二進位檔替換、資料庫遷移和升級後驗證。

## 使用方式

```bash
panguard upgrade
```

## 執行內容

1. **檢查目前版本**與最新版本的差異
2. **下載新的二進位檔**從官方發布頻道
3. **驗證校驗碼**以確保完整性
4. **原地替換二進位檔**
5. **執行遷移**（如果需要資料格式變更）
6. **驗證升級**透過執行快速自我測試

## 範例

<CodeGroup>
```bash 升級至最新版本
panguard upgrade
```
</CodeGroup>
