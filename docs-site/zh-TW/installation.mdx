## 系統需求

| 項目 | 最低需求 |
| --- | --- |
| **作業系統** | macOS 12+、Ubuntu 20.04+、Windows 10+ |
| **Node.js** | >= 18.0.0 |
| **磁碟空間** | 200 MB |
| **記憶體** | 512 MB（Guard 建議 1 GB） |

## 平台支援

不是所有功能在每個平台上支援度都一樣。macOS 和 Linux 是主要開發目標。

| 功能 | macOS | Linux | Windows |
| --- | --- | --- | --- |
| **Skill Auditor** | 完整 | 完整 | 完整 |
| **Scan** | 完整 (lsof, socketfilterfw) | 完整 (ss, ufw/iptables) | 部分 (netstat，無防火牆) |
| **Guard** | 輪詢監控 (5s) | 輪詢監控 (5s) | 輪詢監控 (5s) |
| **Threat Cloud** | 完整 | 完整 | 完整 |

<Warning>
  Windows 支援核心功能，但部分 OS 層級偵測能力不如 macOS 和 Linux。
</Warning>

## 可用 Binary (v1.3.0)

| 平台 | 架構 | 安裝方式 |
| --- | --- | --- |
| macOS | ARM64 (Apple Silicon) | curl 一行安裝或 npm |
| Linux | x64 | curl 一行安裝或 npm |
| Linux | ARM64 | curl 一行安裝或 npm |
| Windows | x64 | npm（建議）或 PowerShell |

## 安裝方式

<Tabs>
  <Tab title="macOS">
    ### 一行指令安裝（建議）

    ```bash
    curl -fsSL https://get.panguard.ai | bash
    ```

    下載 Apple Silicon Mac 的 ARM64 binary 並放進你的 PATH。

    <Tip>
      **Intel Mac 使用者：** curl 安裝預設下載 ARM64 binary。兩個選項：
      1. 改用 npm 安裝：`npm install -g @panguard-ai/panguard`
      2. 先裝 Rosetta 2：`softwareupdate --install-rosetta`
    </Tip>

    ### 用 npm

    ```bash
    npm install -g @panguard-ai/panguard
    ```

  </Tab>

  <Tab title="Linux">
    ### 一行指令安裝（建議）

    ```bash
    curl -fsSL https://get.panguard.ai | bash
    ```

    安裝程式自動偵測架構（x64 或 ARM64）並下載對應的 binary。

    ### 用 npm

    ```bash
    npm install -g @panguard-ai/panguard
    ```

  </Tab>

  <Tab title="Windows">
    ### 用 npm（建議）

    ```powershell
    npm install -g @panguard-ai/panguard
    ```

    ### PowerShell 一行安裝

    ```powershell
    powershell -ExecutionPolicy Bypass -Command "irm https://get.panguard.ai/windows | iex"
    ```

    <Info>
      Windows 建議用 npm 安裝。PowerShell 安裝需要系統管理員權限。
    </Info>
  </Tab>
</Tabs>

## 從原始碼建置

給 contributor 或想跑最新開發版的人：

```bash
git clone https://github.com/panguard-ai/panguard-ai.git
cd panguard-ai
pnpm install
pnpm build

# 跑 CLI
./bin/panguard --help
```

<Info>
  從原始碼建置需要 [pnpm](https://pnpm.io/) 10+ 和 Node.js 20+。
</Info>

## 驗證安裝

安裝完後確認 CLI 可以用：

```bash
panguard --version
```

預期輸出：

```
1.3.0
```

如果找不到指令，確認安裝路徑有加到系統 PATH。npm 全域安裝可以這樣確認：

```bash
npm list -g @panguard-ai/panguard
```

## 解除安裝

<Tabs>
  <Tab title="npm">
    ```bash
    npm uninstall -g @panguard-ai/panguard
    ```
  </Tab>
  <Tab title="curl 安裝">
    ```bash
    rm "$(which panguard)"
    ```
  </Tab>
  <Tab title="原始碼建置">
    直接刪掉 clone 下來的目錄就好。
  </Tab>
</Tabs>

## 設定 AI Agent

安裝完後跑 `panguard setup` 自動設定偵測到的 AI Agent 平台：

```bash
panguard setup
```

自動偵測並設定 Claude Code、Claude Desktop、Cursor、OpenClaw、Codex、WorkBuddy、NemoClaw、ArkClaw 等平台。詳見[快速入門](/quickstart)。
