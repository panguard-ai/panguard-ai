**一行指令。全面防護。無需帳號。**

```bash
curl -fsSL https://get.panguard.ai | bash
```

## 快速開始

安裝、掃描、防護，只需 4 行指令：

```bash
curl -fsSL https://get.panguard.ai | bash
panguard setup
panguard scan --quick
panguard guard start --dashboard
```

`panguard setup` 自動偵測機器上所有 AI 平台（OpenClaw、QClaw、Claude Code、Cursor 等），並注入正確的 MCP 或原生 Skill 設定。

`panguard guard start --dashboard` 啟動 24/7 防護，並在瀏覽器中開啟本地儀表板 `http://127.0.0.1:9100`。

## OpenClaw & QClaw

<Tabs>
  <Tab title="OpenClaw">
    ```bash
    curl -fsSL https://get.panguard.ai | bash && panguard setup --platform openclaw
    ```

    以原生 **Skill** 形式安裝到 `~/.openclaw/skills/panguard/SKILL.md`。重啟 OpenClaw 後：

    ```
    > 審計這個專案的 skills
    > 掃描我的機器是否有漏洞
    > 啟動即時防護
    ```
  </Tab>
  <Tab title="QClaw">
    ```bash
    curl -fsSL https://get.panguard.ai | bash && panguard setup --platform qclaw
    ```

    以 **MCP 伺服器**註冊到 `~/.qclaw/mcp.json`。重啟 QClaw 即可使用 11 個安全工具。
  </Tab>
  <Tab title="自動偵測全部平台">
    ```bash
    curl -fsSL https://get.panguard.ai | bash && panguard setup
    ```

    一行指令自動偵測並配置所有 8 個支援的 AI 平台。
  </Tab>
</Tabs>

## 為什麼選擇 Panguard AI？

傳統安全工具動輒六位數授權費，需要專職團隊。免費工具雖然存在，但缺乏安全工程師幾乎無法使用。

Panguard 採取不同方式：

- **一行指令安裝** -- 不需設定檔、不需調校、不需術語
- **白話警報** -- 透過 Telegram、Slack、Email、LINE 以您的語言接收通知
- **AI 自動判定** -- 系統判斷嚴重性、自動回應、自動報告
- **持續進化** -- 行為基線自適應您的環境
- **技能審計器** -- 安裝前把關每個 AI 技能，阻止供應鏈攻擊
- **100% 免費** -- MIT 授權，無付費版、無功能限制、無付費牆
