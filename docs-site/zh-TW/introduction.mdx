Panguard AI 是一個 100% 開源（MIT 授權）的網路安全平台，為開發者與中小企業帶來企業級端點防護，無需專業安全團隊。不需帳號、不需登入，安裝後即可讓 AI 自動保護您的設備。

核心理念很簡單：**當有事發生，Panguard 通知您。沒事發生時，您什麼都不用做。**

## 為什麼選擇 Panguard AI？

傳統安全工具需要深厚的專業知識、產生大量未分類的警報，且企業授權費用高達六位數。免費工具雖然存在，但沒有專職安全工程師幾乎無法使用。

Panguard AI 採取不同的方式：

- **一行指令安裝** -- 無需設定檔、無需調校、無需專業術語
- **白話通知** -- 透過 Telegram、Slack 或 Email 以您能理解的語言接收警報
- **AI 自動判斷嚴重性** -- 系統自動分類、回應與報告，無需人工介入
- **持續進化** -- 行為基線會自適應您的特定環境

## 快速安裝

```bash
curl -fsSL https://get.panguard.ai | bash
```

然後初始化並執行首次掃描：

```bash
panguard setup
panguard scan --quick
```
