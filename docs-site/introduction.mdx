Panguard AI is an open-source cybersecurity platform that brings enterprise-grade endpoint protection to developers and small businesses -- without requiring a security team. Install the CLI, run setup, and let AI protect your machines automatically. No account required.

The core philosophy is simple: **when something happens, Panguard tells you. When nothing happens, you do nothing.**

## Why Panguard AI?

Traditional security tools require deep expertise, generate thousands of un-prioritized alerts, and cost six figures for enterprise licenses. Free tools exist but are effectively unusable without dedicated security engineers.

Panguard AI takes a different approach:

- **One command install** -- no configuration files, no tuning, no jargon
- **Plain language notifications** -- receive alerts via Telegram, Slack, or Email in language you understand
- **AI auto-judges severity** -- the system triages, responds, and reports without human intervention
- **Gets smarter over time** -- behavioral baselines adapt to your specific environment

## Quick install

```bash
curl -fsSL https://get.panguard.ai | bash
```

Then set up and run your first scan:

```bash
panguard setup
panguard scan --quick
```
