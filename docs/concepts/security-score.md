# Security Score / 安全分數系統

> 一個 0-100 的數字，讓你秒懂系統安全狀態。

---

## 什麼是安全分數？

安全分數是一個 0 到 100 的加權綜合指標，將複雜的安全狀態濃縮成一個直覺的數字和等級。

```
  Score: 85/100 [█████████████████░░░] Grade: A
```

### 等級對照

| 分數   | 等級 | 意義                    |
| ------ | ---- | ----------------------- |
| 90-100 | A    | 優秀 — 防護完善         |
| 80-89  | B    | 良好 — 少數改善空間     |
| 70-79  | C    | 尚可 — 建議處理中等風險 |
| 60-69  | D    | 需注意 — 有顯著風險     |
| 0-59   | F    | 危險 — 需要立即處理     |

---

## 8 個評分因子

安全分數由 8 個加權因子計算，每個因子 0-100 分：

| 因子      | 權重 | 說明                         |
| --------- | ---- | ---------------------------- |
| 防火牆    | 15%  | 防火牆是否啟用、規則是否完善 |
| 開放 Port | 15%  | 不必要的 port 是否暴露       |
| 密碼政策  | 10%  | 是否強制密碼複雜度           |
| 系統更新  | 15%  | OS 和軟體是否為最新版本      |
| 安全工具  | 10%  | 是否安裝防毒/EDR/IDS 等工具  |
| 威脅狀態  | 15%  | 是否有已知威脅活動           |
| 合規程度  | 10%  | 是否符合基本安全框架         |
| 回應能力  | 10%  | 威脅回應機制是否就緒         |

### 計算公式

```
總分 = (防火牆 x 0.15) + (Port x 0.15) + (密碼 x 0.10) +
       (更新 x 0.15) + (工具 x 0.10) + (威脅 x 0.15) +
       (合規 x 0.10) + (回應 x 0.10)
```

---

## 趨勢追蹤

安全分數會追蹤變化趨勢：

| 趨勢      | 條件              | 意義           |
| --------- | ----------------- | -------------- |
| improving | 比上次高 2 分以上 | 安全狀態改善中 |
| declining | 比上次低 2 分以上 | 安全狀態惡化中 |
| stable    | 變化在 2 分以內   | 安全狀態穩定   |

---

## 在哪裡看安全分數

### Panguard Scan

每次掃描結束都會顯示：

```bash
panguard scan
```

```
  ── Risk Score ─────────────────────────

  Score: 72/100 [██████████████░░░░░░] Grade: C
  Trend: improving (+5 since last scan)

  Breakdown:
    Firewall:    80/100
    Open Ports:  60/100
    Passwords:   50/100
    Updates:     90/100
    Tools:       70/100
    Threats:     85/100
    Compliance:  65/100
    Response:    70/100
```

### Panguard Guard

Guard 持續計算即時分數，可透過 `status` 查看：

```bash
panguard guard status
```

### CLI 查詢

透過 `panguard guard status` 即時查看分數和安全狀態。
