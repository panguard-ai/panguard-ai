## Brand Tokens (Quick Reference)

### Colors
| Token | Hex | Usage |
|-------|-----|-------|
| Sage Green | `#8B9A8E` | Primary brand, CTA buttons, active states |
| Deep Charcoal | `#1A1614` | Backgrounds |
| Cream White | `#F5F1E8` | Primary text |
| Status Safe | `#2ED573` | Safe/protected indicators |
| Status Caution | `#FBBF24` | Warning states |
| Status Alert | `#FF6B35` | Alert states |
| Status Critical | `#EF4444` | Critical/danger states |

### Typography
- **Primary**: Modern Sans-Serif (Inter)
- **Mono**: JetBrains Mono (terminal/code)
- Heading: 48pt bold / Subheading: 24pt / Body: 16pt / Caption: 12pt

### Logo Mark
- Interlocking triple-shield icon (represents "collective defense")
- Text: `PANGUARD` [shield icon] `AI`
- Minimum clear space: 1x logo height on all sides
